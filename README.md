# memo.nvim

Neovim上で迅速にマークダウン形式のメモを作成・編集するためのシンプルなプラグインです。

## 主な機能

- **メモの迅速な作成 (`:MemoNew`)**
  コマンドを実行し、ファイル名（または相対パス）を入力することで、指定されたディレクトリ（デフォルトはNeovim起動時のカレントディレクトリ）にマークダウンファイルを作成します。
- **既存ファイルでのアシスト有効化・無効化 (`:MemoEnable` / `:MemoDisable`)**
  すでに開いているファイル（過去のメモなど）に対しても、コマンドを実行するだけで入力アシストを即座にオン／オフにできます。
- **入力アシスト機能**
  - **`<Tab>`**: 
    - **空行で押下時**: 自動的に箇条書きのリストマーカー（`- `）を挿入します。
    - **リスト上で押下時**: その行全体を一段階インデント（段落下げ）します。
    - **表（テーブル）内で押下時**: 次の列へカーソルが自動ジャンプします。
  - **`<Tab>` 連続2回**: 素早く2回押下することで、インデントの代わりに `→` を挿入します。
  - **`<Enter>`**:
    - **リストの継続**: 箇条書き（`- `）や番号付きリスト（`1. `）の行で改行すると、次の行も自動的にリストになります（番号は自動でカウントアップします）。
    - **表の継続**: 表の末尾で改行すると、次の行も自動で表のフォーマット（`| | |`）になります。
    - **`→` のインデント**: `→` が含まれる行で改行した際、次行のテキスト開始位置を自動的に揃えます。
    - ※空のリストや空の表で改行すると、自動継続がキャンセルされます。
- **マークダウンチートシート (`:MemoHelp`)**
  画面右側にマークダウンの構文リファレンスを分割表示します。

## インストールと設定

[lazy.nvim](https://github.com/folke/lazy.nvim) を使用したインストール手順を示します。
インストールと初期設定を同時に行うため、`config` 関数内で `setup()` を呼び出してください。

### 1. lazy.nvim への追加

Neovimのプラグイン設定ディレクトリ（例: `~/.config/nvim/lua/plugins/memo.lua`）に設定ファイルを作成し、以下のコードを追加します。

```lua
return {
  "hamana-dev127/memo.nvim",
  config = function()
    require("memo").setup({
      -- 以下はデフォルト値です。必要に応じて変更してください。
      memo_dir = vim.fn.getcwd(), -- メモのデフォルト保存先ディレクトリ
      default_ext = ".md",        -- デフォルトのファイル拡張子
      auto_show_help = false,     -- メモ利用時にチートシートを自動表示するかどうか
    })
  end,
}
```

### 2. リアルタイムレンダリングの有効化 (LazyVim環境の場合・推奨)

マークダウンの装飾やリストマーカーをリアルタイムに反映させるため、LazyVimの `lang.markdown` エクストラ機能を有効化することを推奨します。

1. Neovim上で `:LazyExtras` を実行します。
2. リストから `lang.markdown` を選択します。
3. `x` キーを押下して有効化（Enable）します。
4. Neovimを再起動して設定を反映させます。

## 使用方法

1. `:MemoNew` を実行し、ファイル名（例: `meeting_notes` や `docs/memo`）を入力してバッファを開きます。
2. 過去のメモなど、既存のファイルでアシストを使いたい場合は、ファイルを開いた状態で `:MemoEnable` を実行します（一時的に機能を止めたい時は `:MemoDisable` を実行します）。
3. 編集モードで `<Tab>` の連続入力や `<Enter>` による自動インデントを活用して記述します。
4. 構文の確認が必要な場合は `:MemoHelp` を実行してチートシートを参照します。
