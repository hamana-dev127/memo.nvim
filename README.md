# 📝 memo.nvim

`memo.nvim` は、Neovimでサクッとメモを取るためのシンプルで賢いプラグインです。
特に講義やミーティング中の素早いメモ取り（マークダウン形式）に特化しており、かゆいところに手が届くアシスト機能がついています。

## ✨ 主な機能

- **コマンド一発でメモ作成 (`:MemoNew`)**
  コマンドを実行して名前を入力するだけで、Neovimを起動したカレントディレクトリにマークダウンファイルを作成して書き始められます。（`docs/memo` のようにディレクトリを指定して作成することも可能です）
- **スマート・キーバインド (Wordのような箇条書き)**
  - **`Tab`**: サクサク段落を下げられます。
  - **`Tab` 2回連続**: 連続でポンポンッと押すと、同じ階層に `→` が自動入力されます。
  - **賢い `Enter`**: `→` のあとに改行すると、次の行の開始位置がピッタリ揃うように自動でインデント（空白）が入ります。
- **安心のチートシート (`:MemoHelp`)**
  「あれ、マークダウンの表ってどう書くんだっけ？」と思った時は、このコマンドを実行してください。画面右側に書き方の早見表が表示されます。

## 🚀 導入手順 (インストール)

このプラグインは [lazy.nvim](https://github.com/folke/lazy.nvim) を使った環境（LazyVimなど）に簡単に導入できます。

### 1. インストール設定
Neovimの設定フォルダ（通常は `~/.config/nvim/lua/plugins/` など）に `memo.lua` というファイルを作成し、以下のコードを貼り付けてください。

```lua
return {
  -- ご自身のGitHubのユーザー名に変更してください
  "your-github-username/memo.nvim",
  
  -- もしローカル（ご自身のPC内）のファイルを使いたい場合は、上の行を消して以下のように書きます
  -- dir = "/Users/naruaki/Library/CloudStorage/GoogleDrive-yoko.na0527@gmail.com/マイドライブ/Programing/Antigravity/ Neovim/memo",
  
  config = function()
    require("memo").setup({
      -- メモの保存先を変更したい場合（デフォルトはNeovim起動時のカレントディレクトリ）
      -- memo_dir = "~/Documents"
    })
  end,
}
```

### 2. リアルタイム・プレビューのおすすめ設定 (LazyVimユーザー向け)

メモを書いている最中に、箇条書きのマークや見出しをリアルタイムで綺麗に表示させるために、以下の設定を強くおすすめします。

1. Neovim上で `:LazyExtras` というコマンドを実行します。
2. リストの中から `lang.markdown` を探してカーソルを合わせます。
3. `x` キーを押して有効化（Enable）します。
4. Neovimを再起動すると、メモアプリがさらに美しく、直感的に使いやすくなります！

## 🛠 設定 (Configuration)

デフォルトの設定は以下の通りです。必要に応じて `setup()` の中で上書きできます。

```lua
require("memo").setup({
  memo_dir = vim.fn.getcwd(), -- メモを保存するデフォルトのディレクトリ（カレントディレクトリ）
  default_ext = ".md",      -- メモファイルのデフォルト拡張子
})
```

## 🎯 使い方まとめ

1. `:MemoNew` と入力してエンター
2. 好きなファイル名（例: `講義メモ`）を入力
3. あとは書くだけ！ `Tab` や `Enter` の便利さを体感してください。
